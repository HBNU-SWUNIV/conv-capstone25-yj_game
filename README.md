# 한밭대학교 컴퓨터공학과 게임하는영재팀

**팀 구성**
- 20222011 최희재
- 20222042 장영아

## <u>Teamate</u> Project Background
- ### 필요성
  - 디지털 네이티브 : 디지털 기기 사용에 익숙한 세대
    - 실제로는 키보드와 같은 전통적인 인터페이스에 대한 활용 능력이 부족
  - 모바일 게임 시장의 확산으로 인한 PC 게임의 위축
    - 조작형 학습 콘텐츠로서의 보드게임형 게임 제안
  
## System Design
  - ### System Requirements
    - 선행 연구 : 키보드 이스케이프 1
    - 각 키가 하나의 방과 연결되는 4*10 크기의 단일맵
      - 몬스터 : 플레이어가 2번 이동 시, 몬스터는 규칙에 따라 1번 추격
      - 키보드 맵과 벽 배치에 따른 경로 탐색 이동 로직이 적용
    - SBERT 기반 AI 가이드 시스템을 사용해 사용자의 자연어 질문에 대해 실시간 의미 기반 응답 제공
  
## Conclusion
  - ### 직관적인 키보드 조작 방식으로 쉬운 PC 접근성 제공
  - ### 향후 GPT 계열 언어 모델을 도입해 완전한 대화형 인터페이스 가이드로 확장 가능
